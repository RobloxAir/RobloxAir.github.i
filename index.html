All About Roblox

Roblox is an online game platform and game creation system developed by Roblox Corporation. It allows users to program games and play games created by other users. Find More on Wikipedia

Initial release date: 1 September 2006


Developer: Roblox Corporation
Publisher: Roblox Corporation
Platforms: Microsoft Windows, Xbox One, Android, iOS, macOS, Fire OS, Macintosh operating systems, Classic Mac OS


Genres: First-person shooter, Massively multiplayer online role-playing game, Nonlinear gameplay, Game creation system


Modes: Single-player video game, Multiplayer video game
